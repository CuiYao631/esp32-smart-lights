# ESP32 智能台灯

ESP32-CAM AI 智能台灯 — 舵机控制系统

## 硬件

| 组件 | 数量 | 说明 |
|------|------|------|
| ESP32-CAM (AI-Thinker) | 1 | 主控 |
| SG90 舵机 | 4 | 关节驱动 |
| WS2812 LED | 24 | 灯头 (暂未启用) |

### 舵机接线

| 舵机 | 功能 | GPIO |
|------|------|------|
| 1号 | 灯头左右转 | 12 |
| 2号 | 上下点头 | 13 |
| 3号 | 力臂关节 | 14 |
| 4号 | 底座 | 15 |

> GPIO 12/13/14/15 为 SD 卡接口引脚，不使用 SD 卡时可安全用于舵机。与摄像头不冲突。

## 依赖库

- **ArduinoJson** (v7+) — Arduino IDE 库管理器搜索安装

## 使用

1. 烧录后 ESP32 会以 AP 模式启动 WiFi
2. 连接热点 `ESP32-SmartLamp`，密码 `12345678`
3. 访问 `http://192.168.4.1`

## API

### `GET /status` — 获取舵机状态

响应示例：
```json
{
  "servos": [
    {"id": 1, "name": "灯头左右", "angle": 0, "pin": 12},
    {"id": 2, "name": "上下点头", "angle": 0, "pin": 13},
    {"id": 3, "name": "力臂关节", "angle": 0, "pin": 14},
    {"id": 4, "name": "底座",     "angle": 0, "pin": 15}
  ]
}
```

### `POST /control` — 控制舵机

**相对旋转**（默认，正值正转，负值反转）：
```json
{"mode": "relative", "servo1": 30, "servo2": -20}
```

**绝对角度**：
```json
{"mode": "absolute", "servo1": 90, "servo3": 45}
```

只需包含要控制的舵机，`servo1` ~ `servo4`，角度范围 0°~180°。

### `POST /reset` — 所有舵机归零
